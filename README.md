### Buienradar app for Homey  
Turn Homey into your personal 'buienradar'.  
With this app Homey will know everything about the expected rain.


##### Speech triggers: 

"Will it rain?"

"Is it going to rain?"

"Is it raining?"

"Will it be dry?"

"Is it dry outside?"
